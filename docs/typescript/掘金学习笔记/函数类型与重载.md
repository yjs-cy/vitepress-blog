# 函数
函数用来封装那些重复的逻辑，更好地组织代码的逻辑结构。
## 函数的两种写法
声明式：允许在声明之前调用，表达式必须要在其后调用
```typescript
// 表达式
const handler = function (args) {}; // 函数表达式
const handler = (args) => {}; //  箭头函数表达式

// 声明式
function handler(args) { }; // 函数声明式
```
## 函数的类型校验
真正关注的也就只有入参和返回值。  
参数后标注这个参数的类型，在参数块与函数体之间标注函数的返回值类型。
```typescript
// 表达式类型校验
const sum = function(a: number, b: number): number {
  return a + b;
};

// 声明式类型校验
function sum(a: number, b: number): number {
  return a + b;
}

// 类型别名写法
type Sum = (a: number, b: number) => number;
const sum: Sum = function(a, b) {
  return a + b;
};
```
### 无返回值的写法
在 JavaScript 中，如果没有显式的 return 语句，这个函数的执行结果会是 undefined  
在 TypeScript 中，我们需要将这个函数的返回值类型标注为 void 而不是 undefined
```typescript
function handler1(): void {};
```
## 函数重载
根据不同的入参匹配不同的实际逻辑，实现一个函数名走天下。
```typescript
// 简单案例
function sum(base: number, incre: number): number;
function sum(baseArray: number[], incre: number): number[];
function sum(incre: number, baseArray: number[]): number[];
function sum(baseArray: number[], increArray: number[]): number[];
function sum(x: number | number[], y: number | number[]): number | number[] { }
```
