# 类型别名、联合类型、交叉类型

## 类型别名
```typescript
type Handler = () => void;

type User =  {
  userName: string;
  userAge: number;
  userMarried: boolean;
  userJob?: string;
}
```
## 什么是字面量类型？
```typescript
const fixedStr: 'linbudu' = 'linbudu'; // 值只能是 'linbudu'
const fixedNum: 599 = 599; // 值只能是 599
```
## 联合类型（逻辑或）
在实际应用中，最常见的应该是字面量联合类型，它表示一组精确的字面量类型
```typescript
type PossibleTypes = string | number | boolean;
type Status = 'success' | 'failure';
type Code = 200 | 404 | 502;
```
由接口组成的联合类型
```typescript
interface VisitorUser {}
interface CommonUser {}
interface VIPUser {}
interface AdminUser {}

type User = VisitorUser | CommonUser | VIPUser | AdminUser;

const user: User = {
  // ...任意实现一个组成的对象类型
}
```
## 交叉类型（逻辑与）
```typescript
interface UserBasicInfo {}
interface UserJobInfo {}
interface UserFamilyInfo {}
type UserInfo = UserBasicInfo & UserJobInfo & UserFamilyInfo;
// 上面的代码可以这样理解：一个新的类型内部合并了这几个对象类型
type UserInfo = {
  ...UserBasicInfo,
  ...UserJobInfo,
  ...UserFamilyInfo
}
```
